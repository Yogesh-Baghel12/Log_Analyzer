# Log Analyzer

A simple web-based log analyzer built with Python, Langchain and JavaScript.

## Features
- Upload and analyze log files
- Interactive web interface
- Visualize and filter log data

## Project Structure
```
log_analyzer/
  app.py              # Main backend application (Flask or similar)
  requirements.txt    # Python dependencies
  static/
    index.html        # Main HTML page
    script.js         # Frontend JavaScript
    style.css         # Stylesheet
```

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone "https://github.com/Yogesh-Baghel12/Log_Analyzer.git"
   cd log_analyzer
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   python app.py
   ```

4. **Open your browser:**
   Navigate to `http://localhost:5000` to use the Log Analyzer.

## Requirements
- Python 3.7+
- See requirements.txt for Python packages
